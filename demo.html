Funcionalidades Implementadas

  • Sistema de autenticação completo (login/cadastro)
  • Catálogo de produtos integrado com DummyJSON API
  • Carrinho de compras com controle de quantidade
  • Sistema de pagamento (PIX, cartão crédito/débito)
  • Gestão de pedidos com status e histórico
  • Gestão de endereços (até 3 endereços)
  • Sistema de logística e rastreamento
  • Interface responsiva e moderna
  • Validações de segurança e sessão
  • Integração com ViaCEP para endereços

Como Testar

  1. Login: Use as credenciais do DummyJSON ou crie uma nova conta
  2. Produtos: Navegue pelo catálogo e adicione produtos ao carrinho
  3. Carrinho: Ajuste quantidades e prossiga para o checkout
  4. Endereço: Adicione um endereço (integração com ViaCEP)
  5. Pagamento: Teste PIX ou cartões com os dados fornecidos
  6. Pedidos: Acompanhe o status e histórico dos pedidos
Importante: Este é um sistema de demonstração. Todos os dados são simulados e não há integração real com sistemas de pagamento.
